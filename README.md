# Paste App

A responsive Paste Management Web Application built using React, Redux Toolkit, and Tailwind CSS. The application allows users to create, edit, delete, copy, search, and manage text snippets efficiently with local storage persistence.

---

## Features

- Create and save text pastes
- Edit existing pastes
- Delete pastes
- Copy paste content to clipboard
- Search functionality
- Local storage persistence
- Responsive UI using Tailwind CSS
- State management using Redux Toolkit

---

## Technologies Used

- React.js
- Redux Toolkit
- Tailwind CSS
- JavaScript
- Vite
